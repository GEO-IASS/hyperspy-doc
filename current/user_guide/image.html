Image tools¶

These methods are only available for Signal object with signal_dimension equal to two.

Image registration (alignment)¶

New in version 0.5.

The align2D() method provides advanced image alignment functionality, including subpixel alignment.

Cropping an image¶

In addition to cropping using the powerful and compact Indexing the following method is available to crop spectra the familiar top, bottom, left, right syntax.

  • crop_image()
