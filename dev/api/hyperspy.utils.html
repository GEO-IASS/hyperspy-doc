hyperspy.utils.markers module¶

Markers that can be added to Signal plots.

Example¶

>>> import scipy.misc
>>> im = hs.signals.Image(scipy.misc.lena())
>>> m = hs.plot.markers.rectangle(x1=150, y1=100, x2=400, y2=400, color='red')
>>> im.add_marker(m)

hyperspy.utils.plot module¶
