mlpca Module¶

hyperspy.mva.mlpca.mlpca(X, varX, p, convlim=1e-10, maxiter=50000)¶

This function performs MLPCA with missing data.

Arguments: X is the mxn matrix of observations. stdX is the mxn matrix of standard deviations

associated with X (zeros for missing measurements).

p is the model dimensionality.

Returns: U,S,V are the pseudo-svd parameters. Sobj is the value of the objective function. ErrFlag indicates exit conditions:

0 = nkmal termination 1 = max iterations exceeded.
